# Migration Plan

1. Inventory the existing HelpDocs articles.
2. Group them into handbook sections.
3. Recreate each article as Markdown or MDX.
4. Verify links, images, and search behavior.
5. Point `handbook.hallscloud.org` at the deployed static site.

## Initial structure

- `docs/` for handbook content
- `src/pages/` for the landing page
- `src/css/` for branding and layout styling

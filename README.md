# Hallscloud Handbook

This repository contains the Docusaurus site that will host the handbook for `hallscloud.org` at `handbook.hallscloud.org`.

## Local development

1. Install dependencies.
2. Run `npm run start` to preview locally.
3. Run `npm run build` before publishing.

## Content migration

The current HelpDocs content should be migrated into `docs/` article by article. Start with the highest-traffic help pages, then move internal reference material, then clean up redirects and broken links.

## Deployment

This site is set up as a static Docusaurus handbook and can be deployed to GitHub-backed hosting or any static host that serves `handbook.hallscloud.org`.
